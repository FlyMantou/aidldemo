智能双排键aidl远程接口服务示例
===========================
该示例程序用于测试与“智能双排键控制面板”程序提供的AIDL远程服务接口，该示例为安卓程序，使用'Android Studio3.0.1'开发。

##使用说明

    1.将aidl文件夹复制 到自己项目的main文件夹下

    2.make project

    3.根据文档中的调用示例进行接口注册

    4.即可使用
